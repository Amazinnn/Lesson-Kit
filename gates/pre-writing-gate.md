# Gate: Pre-Writing Gate

Run this before final lesson prose.

Pass only if required source, analysis, and plan files exist for the task; PDF scope is visually verified when needed; knowledge inventory is complete enough for scope; subject-required check is done; body details have updated or confirmed the knowledge inventory; feedback is preserved before diagnosis; high-impact uncertainty is confirmed or marked; lesson/problem-set division exists; and lesson structure/rendering plans exist.

Required files must be explicit artifacts, not private reasoning or placeholder stubs.

| Failure | Return to | Forbidden repair |
|---|---|---|
| Source scope missing or uncertain | `intermediate/lesson_loop/01_inputs/source-inventory.md` | Claiming full coverage in final prose |
| PDF scope or visual reading not verified | `intermediate/lesson_loop/01_inputs/pdf-reading-log.md` | Using text extraction as source of truth |
| Knowledge inventory shallow or missing | `intermediate/lesson_loop/02_analysis/chapter-knowledge-inventory.md` | Starting final lesson prose and hoping to cover gaps later |
| Subject-required content unchecked | `intermediate/lesson_loop/02_analysis/subject-required-check.md` | Treating textbook headings as the complete knowledge system |
| Body details not integrated | `intermediate/lesson_loop/02_analysis/textbook-body-detail-extraction.md` | Adding a loose “notes” section at the end |
| Feedback diagnosed before preservation | `intermediate/lesson_loop/01_inputs/exercise-feedback-record.md` | Reconstructing “original feedback” from the diagnosis |
| Lesson/problem-set boundary missing | `intermediate/lesson_loop/03_plans/lesson-problem-set-decision.md` | Letting examples and drills accumulate inside the lesson |
| Structure or rendering plan missing | `intermediate/lesson_loop/03_plans/lesson-structure-plan.md` or `intermediate/lesson_loop/03_plans/lesson-rendering-plan.md` | Formatting final prose after the fact without a plan |

Failure must return to the earliest broken layer.
