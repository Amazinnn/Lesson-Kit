# Gate: Answer Key Export Gate

The answer key passes only if every problem has a corresponding answer, numbering/order/tags match the problem set, explanations match the requested depth, and changes to the problem set are reflected in the key.

If any item is missing or misaligned, return to `03_plans/answer-key-sync-plan.md`.

Fail if the answer key was not regenerated or checked after changes to problem wording, numbering, order, options, source data, figures, LaTeX, or formatting that can affect answers.
