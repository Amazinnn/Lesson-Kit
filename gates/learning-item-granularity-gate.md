# Gate: Learning Item Granularity

Check that first-pass learning items are concrete knowledge-point tasks.

| Failure | Return to | Forbidden repair |
|---|---|---|
| Item is a whole-section heading | `intermediate/first_pass/02_analysis/learning-item-granularity-analysis.md` | Keeping the heading and adding a vague question |
| Item is an isolated word or symbol | `intermediate/first_pass/02_analysis/learning-item-granularity-analysis.md` | Treating a glossary entry as a task |
| Item likely exceeds 8 minutes | `intermediate/first_pass/02_analysis/learning-item-granularity-analysis.md` | Telling the student to spend more time |
| Ordinary section has more than 10 items | `intermediate/first_pass/02_analysis/learning-item-priority-map.md` | Making all items required |
| Complex section has more than 13 items | `intermediate/first_pass/02_analysis/learning-item-priority-map.md` | Hiding overflow items in notes |
| Complex section has too many required items | `intermediate/first_pass/03_plans/first-batch-suggestion-plan.md` | Forcing a mandatory batch |
| Item is too narrow for scene-judgment MCQ (pure definition/term/condition without application contrast) | `intermediate/first_pass/02_analysis/kp-consolidation-analysis.md` | Keeping the item as an independent KP with a trivial MCQ |
