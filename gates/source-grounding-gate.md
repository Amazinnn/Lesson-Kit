# Gate: Source Grounding

Every first-pass learning item, checkpoint question, MCQ, and checking hint must point back to a source location.

| Failure | Return to | Forbidden repair |
|---|---|---|
| Item has no source locator | `intermediate/first_pass/02_analysis/candidate-learning-item-inventory.md` | Adding broad section names without exact locator |
| Source location cannot support the reading task | `intermediate/first_pass/02_analysis/candidate-learning-item-inventory.md` | Rephrasing the task to sound source-based |
| MCQ answer cannot be checked from source | `intermediate/first_pass/03_plans/checkpoint-question-final-plan.md` | Writing an explanatory answer from memory |
| PPT/PPTX source item lacks slide-image locator | `intermediate/first_pass/01_inputs/ppt-visual-reading-log.md` | Citing copied slide text or extracted XML |
