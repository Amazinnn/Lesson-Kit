# Gate: No Summary Substitution

Use this gate before exporting any source-guided first-pass lesson.

| Failure | Return to | Forbidden repair |
|---|---|---|
| Final artifact can be read without returning to source material | `first-pass-structure-plan.md` | Removing a few details while preserving self-contained summary structure |
| Knowledge point cards explain source content as a substitute for reading | `knowledge-points.md` and final artifact | Renaming explanations as reading guides |
| Short explanations omit source locator or reading task | `knowledge-points.md` | Adding source locator only at section level |
| Checkpoint questions ask for memorized final prose instead of source-grounded understanding | `knowledge-points.md` and `coverage-check.md` | Adding generic “why” questions |

Pass only when the final artifact pushes the student back to source material for full understanding.
