# lesson-kit

**Modular study material generator.**

Extracts knowledge points and exercises from source materials (PDF, PPT, etc.), then assembles them into configurable learning formats: first-pass overviews, in-depth lessons, problem sets, and more.

Built around a **knowledge-point pool + question pool** architecture — extract once, render in any form.

## Philosophy

- **Low entry barrier** — minimal cognitive overhead to start a study session
- **Source-guided** — output leads students back to the source, never replaces it
- **Modular views** — the same knowledge foundation powers different learning formats
- **Pedagogy-driven** — design decisions grounded in educational theory (cognitive load, testing effect, advance organizers)

## Status

Pre-release. Migrating from the original V17 workflow. Core extraction pipeline works; pool architecture and multi-view rendering under design.
