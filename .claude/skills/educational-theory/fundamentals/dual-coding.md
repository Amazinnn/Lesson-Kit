# Dual Coding & Multimedia Learning

## Core Claim

Information presented through both verbal and visual channels is encoded more robustly than information through either channel alone — because the brain creates two independent memory traces.

## Key Research

Paivio (1986) proposed Dual Coding Theory. Mayer (2001–2025) extended it into the Cognitive Theory of Multimedia Learning (CTML) with 12 evidence-based principles: multimedia (words+pictures > words alone), contiguity (place related items together), modality (narration+animation > text+animation), and redundancy (avoid duplicating the same information across channels).

## Effect Size & Confidence

- Multimedia effect: d = 1.00–1.50 (high confidence for retention)
- Contiguity principle: d = 0.70–1.20 (high confidence)
- Modality effect: d = 0.60–1.00 (high confidence)
- Redundancy principle: d = 0.50–0.80 (medium-high confidence)

## Boundary Conditions

- Dual coding helps most when both channels carry complementary information.
- When channels carry redundant information (same content in text AND narration), learning may suffer (redundancy effect).
- Effects depend on learner prior knowledge and pacing.

## Practical Implications

- In 速览, formulas + prose = dual coding (symbolic + verbal). The through-line accent provides the verbal framing.
- Mermaid diagrams in 速览 are effective only if they add information not present in the text.
- Avoid text-only walls: every block of text longer than 250 Chinese characters should include a formula, code block, table, or diagram.

## Connection to Other Theories

- **Cognitive Load Theory** (see fundamentals/cognitive-load-theory.md) provides the cognitive mechanism: dual coding reduces cognitive load by distributing processing across two channels.
- **Signaling Principle** (part of CTML) connects to how through-line accents guide attention.
