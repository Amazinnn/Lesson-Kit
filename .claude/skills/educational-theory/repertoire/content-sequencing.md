# Sequencing Content

## 一句话定位

呈现顺序不是"目录"，而是"认知路径"——每一步应该为下一步做准备。

## Related Theories

- **Cognitive Load Theory** (see fundamentals/cognitive-load-theory.md): Sequence from simple to complex to manage intrinsic load. Each step should build on schemas formed in previous steps.
- **Progressive Differentiation** (Ausubel): Start with the most inclusive, general ideas, then progressively differentiate into specifics. Top-down, not bottom-up.
- **Spiral Curriculum** (Bruner): Key concepts should be revisited at increasing levels of complexity across the curriculum. The 速览's through-line is a within-chapter spiral.
- **Elaboration Theory** (Reigeluth): Simple → Complex, not Chronological. Teach the simplest complete version of a task first, then add complexity.

## Teaching Application

- Default: preserve source order (it was designed by domain experts for a reason).
- Break source order only when there's a clear pedagogical dependency chain (concept A must precede concept B, but the source puts B first for narrative reasons).
- Within each section: definitions → tools → applications → exceptions. This mirrors the natural learning progression.
- If a knowledge point's prerequisite is not yet established, either resequence or add a brief anchor.

## Aha Moment

> Source order is usually correct for expert readers who already know the domain. For novice readers, pedagogical order (prerequisites first, then new content) may differ. The tension between "source order" and "pedagogical order" is where instructional design adds value.
