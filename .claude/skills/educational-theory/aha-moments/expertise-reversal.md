# The Expertise Reversal Effect

Instructional techniques that help novices (worked examples, detailed scaffolding) can hinder experts, while techniques that help experts (minimal guidance, problem-solving from scratch) can overwhelm novices.

## Why This Matters

There is no single "best" teaching method — it depends on the learner's prior knowledge. For a first-pass 速览 (novice audience), worked examples and explicit scaffolding are appropriate. For later revision materials (more expert audience), minimal guidance and open-ended problems become more effective. The same content, same format — wrong for the wrong audience.

## Source

Kalyuga, Ayres, Chandler & Sweller (2003). The expertise reversal effect. *Educational Psychologist*, 38(1), 23–31.
