# Problem Source Consistency Check

| Source problem | Final problem location | Numbering preserved? | Wording/data verified visually? | Figure/table dependency handled? | Pass/Fail | Repair return file |
|---|---|---|---|---|---|---|

## Minimum Completion Standard

This check fails if it does not compare source and final problem at the level of numbering, wording, data, subquestions, figure/table dependency, and section placement. Source exercises dependent on figures must be self-contained or explicitly marked unresolved.
