# Skill: Lesson Structure Planning

Final lesson structure must be anchored to source knowledge, subject requirements, priority, and dependency. It must not be anchored to feedback wording, wrong-answer IDs, or generated problem types.

The plan should decide chapter/section preservation, local reorganization, module priorities, dependencies, embedded examples, diagrams/tables/code/formulas, and which content belongs in the separate problem set.

If revising an existing lesson, preserve useful existing structure unless it conflicts with source coverage, subject requirements, or feedback-derived knowledge needs.
