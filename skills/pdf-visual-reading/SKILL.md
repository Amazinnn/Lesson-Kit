# Skill: PDF Visual Reading

For PDF/image-based textbook work, visually verify the requested scope. Metadata and text extraction may help locate pages, but final claims about content, formulas, tables, figures, exercises, and numbering must be visually verified.

Create `01_inputs/pdf-reading-log.md` with page range, visual-reading status, skipped pages if any, figures/tables requiring attention, exercise-numbering notes, and unresolved source problems.

Do not use scripts or OCR as the source of truth. If a table, figure, diagram, formula, or exercise appears only in image form, inspect it visually.
