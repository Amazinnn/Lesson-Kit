# Skill: Scannable Markdown Layout

Dense materials must be easy to scan. If a knowledge block exceeds roughly 250–300 Chinese characters without bold anchors, lists, formulas, code, tables, or diagrams, revise it.

Use paragraphs for explanations, bold leads for local anchors, lists for three or more parallel items, tables for concise comparison, code blocks for implementations, and Mermaid for useful structure/process/dependency.

Avoid two extremes: fragmented micro-bullets that break thinking continuity, and dense prose walls where definitions, formulas, conditions, steps, and code points cannot be found quickly.
