# Skill: Source And Scope

Record sources before analysis. The source inventory must state the requested scope, source type, page/section range when available, included exercises, included examples/thinking questions, existing lesson files, student feedback files, missing or uncertain sources, and whether the task claims full coverage.

Do not diagnose or rewrite here. If source scope is uncertain, stop full-coverage claims until the scope is locked.

Recoverable gaps may be marked and scoped out. Blocking gaps include unknown requested chapter range, missing pages in requested scope, unexplained exercise numbering jumps, or conflict between user-stated page count and visible source.
