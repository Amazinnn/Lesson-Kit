# Skill: Subject Data Structures

For data-structure materials, check the full chain: ADT semantics, logical structure, storage representation, invariant, core operations, process trace, boundary case, time complexity, space complexity, C/C-like implementation or close pseudocode, and neighboring structure comparison.

Operation-heavy topics should show state changes concisely: pointers in linked lists, `front`/`rear`/`size` in queues, stack contents in expression processing, BST search constraints, heap index movement, recursion intervals, and parent-array changes in union-find.
