# Skill: Problem Set Selection

Select problems by coverage, source consistency, and training value. Preserve source wording and numbering for textbook-based sets. When full coverage is requested, include all relevant source exercises; otherwise deduplicate homogeneous problems and add variants only when coverage or transfer requires them.

Each selected or generated problem must map to one or more lesson knowledge points and training targets.

Before filtering source exercises, solve or answer-model candidate problems enough to know their knowledge target, model, condition signature, technique, boundary, representation, common lure, and training value. Do not delete a problem as homogeneous based only on surface wording.

A generated supplemental problem is allowed only for a documented in-scope coverage or transfer gap and should be labeled as added/generated according to the artifact style.
