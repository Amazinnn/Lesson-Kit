# Skill: Subject Required Content

Check what the subject normally requires before writing. Do not omit a required form merely because the source or feedback did not explicitly mention it.

Algorithm analysis: problem size, time complexity, space complexity, best/average/worst cases, loop counts, summations, recurrences, asymptotic notation, dominant terms, tight/upper/lower bound distinctions.

Data structures: ADT/logical semantics, storage representation, invariant, operation process, C/C-like implementation or close pseudocode, boundary cases, time/space complexity, neighboring structures.

Discrete mathematics: definitions, symbols, formulas, theorems, equivalent transformations, conditions, boundary cases, derivation/proof routes, recognition in statements.

University physics: definitions, laws/theorems, physical models, formulas, quantity meanings, units, directions, signs, reference frames, coordinates, applicability conditions, derivation conclusions, classic cases, failure conditions.
