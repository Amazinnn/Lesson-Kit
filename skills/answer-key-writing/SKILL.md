# Skill: Answer Key Writing

Mirror the problem set. For each problem, give enough reasoning to show key judgments, knowledge calls, operations, conditions, and final result. Keep wording synchronized with problem numbering and tags.

If the user requested a terse key, keep explanations concise but still avoid ambiguous unsupported final answers.

Use the same language as the problem set unless the user requests otherwise. If the problem set is English, the answer key should normally be English; if it is Chinese, the answer key should normally be Chinese.
