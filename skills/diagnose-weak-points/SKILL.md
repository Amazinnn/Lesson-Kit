# Skill: Diagnose Weak Points

Diagnose feedback at the weak-module level before revising lesson body. For each feedback item, distinguish surface issue, underlying knowledge detail, wrong mental model, dependency break, neighboring problem forms affected, and whether the issue is conceptual, procedural, representational, transfer-related, reading-related, computational, or careless.

Do not turn every feedback item into a new lesson section. Do not build the lesson skeleton from feedback wording.

Ask confirmation only for high-impact uncertain diagnoses that would change major structure, create independent knowledge points, or require large practice repair. If direct confirmation is not possible and the user wants final output, mark uncertainty and use conservative placement.
